# Student Performance Analysis

## Project Description

This is a beginner-level project created to practice Jupyter Notebook, Python, Pandas, and Matplotlib. The project analyzes student marks, calculates average scores, identifies top-performing students, and visualizes the results using charts.

This project was developed for learning and practice purposes while exploring data analysis with Python and Jupyter Notebook.

## Learning Objectives

* Learn how to use Jupyter Notebook
* Work with CSV files using Pandas
* Perform basic data analysis
* Create visualizations using Matplotlib
* Organize and document a project for GitHub

## Features

* Load student data from a CSV file
* Calculate average marks
* Find the top-performing student
* Generate bar charts and pie charts
* Display summary statistics

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
